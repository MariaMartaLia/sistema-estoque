# 01 - Visão Geral do Projeto

## 1. Objetivo

O Sistema de Gestão de Estoque foi desenvolvido para controlar entradas e saídas de produtos, garantindo integridade dos dados,controle de concorrência e aplicação de regras de negócio.
O sistema foi projetado para pequenos negocios estabelecimentos que necessitam de controle preciso de estoque, rastreabilidade de movimentações.

 ---

 ## 2. Problema Resolvido
 - Falta de controle de estoque;
 - Inconcistência de dados;
 - Duplicidade de operações simultâneas;
 - Ausência de rastreabilidade;
 - Estoque negativo;

 ---

 ## 3. Escopo de Versão 1.0

 - Cadastro de produtos 
 - Listagem de produtos
 - Movimentação de entrada
 - Movimentação de saída
 - Controle transacional
 - Bloqueio de concorrência 
 - Interface web integrado

 ---

 ## 4. Tecnologias Utilizadas

 - Ubunto
 - PostgreSQL
 - Python 3
 - FastAPI
 - React + Vite
 - Git / Github
